# Sports Challenge Telegram Bot

Telegram-бот для челленджа тренировок: участники отмечают прогресс **прямо в группе**, а бот отвечает *похвалой* на фото/скрин **с подписью**.

## Как отмечать прогресс в группе

- **Отправьте фото/скрин в группу**.
- **Добавьте подпись** (что за тренировка/активность).
- Бот ответит похвалой. Если подписи нет — попросит добавить.

## Команды

- `/start` — краткая помощь

## Переменные окружения

Смотри `SETUP_GUIDE.md` для пошаговой настройки. Минимальный набор:

- `BOT_TOKEN`
- `GROUP_CHAT_ID` (опционально; если задан — бот отвечает только в этой группе)

## Локальный запуск (быстро)

1. Установи зависимости:

```bash
pip install -r requirements.txt
```

2. Создай `.env` и заполни переменные окружения (см. выше).
3. Запусти бота:

```bash
python bot.py
```
