# Sports Challenge Telegram Bot

Telegram-бот для челленджа тренировок: участники отмечают прогресс **прямо в группе**, а бот логирует отметки в Google Sheets.

## Как отмечать прогресс в группе

- **Просто отправьте фото в группу** (любое фото тренировки).
- Бот ответит подтверждением и запишет ✅ в Google Sheets.
- Повторная отметка в тот же день не дублируется (1 раз в день на пользователя).

## Команды

- `/start` — регистрация (в личных сообщениях с ботом)
- `/cancel` — отмена регистрации
- `/stats` — сколько людей отметилось сегодня

## Переменные окружения

Смотри `SETUP_GUIDE.md` для пошаговой настройки. Минимальный набор:

- `BOT_TOKEN`
- `GOOGLE_SHEET_ID`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
- `GOOGLE_REFRESH_TOKEN`
- `GROUP_CHAT_ID`
- `GROUP_INVITE_LINK` (опционально)
- `DAILY_NOTIFY_TIME` (опционально)
- `TIMEZONE` (опционально)

## Локальный запуск (быстро)

1. Установи зависимости:

```bash
pip install -r requirements.txt
```

2. Создай `.env` и заполни переменные окружения (см. выше).
3. Запусти бота:

```bash
python bot.py
```
